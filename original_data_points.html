Retrieve original data points¶

First, let’s check we can retrieve the original data points consistently.

Behind-the-scenes¶

We start by loading SPARQL queries, and sends them to RDFox to be answered against the pre-converted data. For simplicity the details are hidden in a separate Python file:

from example_data import results

Prodcom¶

We can retrieve original data points from Prodcom linked to a classification code (08121230) using the following SPARQL query:

PREFIX : <https://ukfires.org/probs/ontology/>
PREFIX time: <http://www.w3.org/2006/time#>
PREFIX gn: <http://www.geonames.org/ontology#>
PREFIX quantitykind: <http://qudt.org/vocab/quantitykind/>
PREFIX ufpc: <https://ukfires.org/probs/ontology/prodcom/>

SELECT DISTINCT ?ObjectName ?Role ?Value ?Dataset ?Year ?Region ?Metric
WHERE {
    ?Object :objectName ?ObjectName ;
        :hasClassificationCode [ :codeName "08121230" ] .
    ?Observation :objectDefinedBy ?Object ;
        :hasTimePeriod [ time:unitType time:unitYear ; time:year ?Year ] ;
        :hasRegion [ gn:name ?Region] ;
        :hasRole ?Role ;
        :partOfDataset ?Dataset ;
        :metric ?Metric ;
        :measurement ?Value .
}
ORDER BY ?Object ?Year

results["original_prodcom"]
ObjectName Role Value Dataset Year Region Metric
0 PRODCOM Object from Code 08121230 :SoldProduction 60255156000 :prodcom/PRODCOM2017DATA 2017 United Kingdom quantitykind:Mass
1 PRODCOM Object from Code 08121230 :SoldProduction 68391115000 :prodcom/PRODCOM2018DATA 2018 United Kingdom quantitykind:Mass

BGS Minerals Yearbook¶

If we want to retrieve the data points specific of the BGS Minerals Yearbook dataset:

Fig. 4 All the initial data points from BGS.¶

We can use the following query:

PREFIX : <https://ukfires.org/probs/ontology/>
PREFIX time: <http://www.w3.org/2006/time#>
PREFIX gn: <http://www.geonames.org/ontology#>
PREFIX quantitykind: <http://qudt.org/vocab/quantitykind/>
PREFIX ufpc: <https://ukfires.org/probs/ontology/prodcom/>

SELECT DISTINCT ?ObjectName ?Role ?Value ?Year ?Region ?Metric
WHERE {
    ?Observation :partOfDataset ufpc:BGSMineralsYearbook2015 ;
        :objectDirectlyDefinedBy [ :objectName ?ObjectName ] ;
        :hasTimePeriod [ time:unitType time:unitYear ; time:year ?Year ] ;
        :hasRegion [ gn:name ?Region] ;
        :hasRole ?Role ;
        :metric ?Metric ;
        :measurement ?Value .
}
ORDER BY ?Object ?Year
