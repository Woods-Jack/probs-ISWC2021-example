Inferred observations¶

Let’s query for all the observations that would be relevant to modelling production of two object types, CrushedStone and SandAndGravel, in all years that are available.

Fig. 12 All the observations relevant to Aggregates components (CrushedStone and SandAndGravel).¶

Here is the SPARQL query:

PREFIX : <https://ukfires.org/probs/ontology/>
PREFIX gn: <http://www.geonames.org/ontology#>
PREFIX time: <http://www.w3.org/2006/time#>

SELECT ?Observation ?Object ?Year ?Bound ?Value
WHERE {
  ?Observation :objectDefinedBy [ :objectName ?Object ] ;
               :hasRegion [ gn:name "United Kingdom" ] ;
               :hasRole :SoldProduction ;
               :hasTimePeriod [ time:year ?Year ] ;
               :bound ?Bound ;
               :measurement ?Value .
  FILTER( ?Object = "Crushed stone" || ?Object = "Sand & Gravel" )
  FILTER( ?Year = "2014"^^xsd:gYear || ?Year = "2018"^^xsd:gYear )
}
ORDER BY ?Object ?Bound ?Year

We can see where these values have come from:

PREFIX : <https://ukfires.org/probs/ontology/>
PREFIX prov: <http://www.w3.org/ns/prov#>
PREFIX gn: <http://www.geonames.org/ontology#>
PREFIX time: <http://www.w3.org/2006/time#>

SELECT DISTINCT ?Observation ?WDFObject ?WDFValue
WHERE {
  ?Observation :objectDefinedBy [ :objectName ?Object ] ;
               :hasRegion [ gn:name "United Kingdom" ] ;
               :hasRole :SoldProduction ;
               :hasTimePeriod [ time:year ?Year ] ;
               prov:wasDerivedFrom ?WDF .

  FILTER( ?Object = "Crushed stone" || ?Object = "Sand & Gravel" )
  FILTER( ?Year = "2014"^^xsd:gYear || ?Year = "2018"^^xsd:gYear )

  ?WDF a :DirectObservation ;
         :objectDirectlyDefinedBy [ :objectName ?WDFObject ] ;
         :measurement ?WDFValue
}
ORDER BY ?Observation ?WDFObject

Further aggregation¶

We can further query for the aggregates observations of Aggregates:

PREFIX : <https://ukfires.org/probs/ontology/>
PREFIX gn: <http://www.geonames.org/ontology#>
PREFIX time: <http://www.w3.org/2006/time#>

SELECT ?Year ?Bound ?Value
WHERE {
  ?Observation :objectDefinedBy [ :objectName ?Object ] ;
               :hasRegion [ gn:name "United Kingdom" ] ;
               :hasRole :SoldProduction ;
               :hasTimePeriod [ time:year ?Year ] ;
               :bound ?Bound ;
               :measurement ?Value .
  FILTER( ?Object = "Aggregates" )
  FILTER( ?Year = "2014"^^xsd:gYear || ?Year = "2018"^^xsd:gYear )
}
ORDER BY ?Bound ?Year

This results in:
